# Estrutura e Hierarquia

---

Viktor moldou seu grupo à imagem do que aprendeu na Rússia: células pequenas, independentes, conectadas apenas pelo topo.

## Organograma

```
              ┌─────────────┐
              │   PAKHAN    │ ← Viktor Petrov
              └──────┬──────┘
                     │
         ┌───────────┼───────────┐
         ▼                       ▼
   ┌───────────┐         ┌────────────┐
   │ SOVETNIK  │         │  BRIGADIR  │ (x2-3)
   └───────────┘         └──────┬─────┘
                                │
                     ┌──────────┼──────────┐
                     ▼                     ▼
               ┌───────────┐        ┌───────────┐
               │   BOETS   │        │   BOETS   │ (x3-5)
               └─────┬─────┘        └───────────┘
                     │
              ┌──────┴──────┐
              │ SHESTYORKA  │ (vários)
              └─────────────┘
```

## Cargos

| Posição | Função |
|---------|--------|
| **Pakhan** | Líder absoluto. Todas as decisões estratégicas. |
| **Sovetnik** | Conselheiro. Estratégia e conflitos internos. Substitui o Pakhan. |
| **Brigadir** | Líder de célula. Comanda 3-5 homens em operações específicas. |
| **Boets** | Soldado. Executa: coletas, entregas, vigilância, violência. |
| **Shestyorka** | Associado. Trabalhos menores sem conhecer a estrutura. Descartável. |

## Regras Internas

1. **Compartimentalização** — Ninguém conhece mais do que precisa.
2. **Sem cooperação com autoridades** — Quem fala, morre.
3. **Obshchak** — Porcentagem de todo lucro vai para o fundo comum.
4. **Resolução interna** — Conflitos resolvidos pelo Sovetnik. Nunca na rua.
5. **Sem drogas no varejo** — Proibição absoluta.

## Tamanho

~20 membros ativos em 3-4 células. Deliberadamente pequeno.
